# คู่มือพัฒนาเครื่องมือประเมิน DISC สำหรับพยาบาลไทย

## บทนำ

คู่มือนี้ออกแบบเพื่อพัฒนาแบบประเมิน DISC สำหรับพยาบาลไทย โดยเน้นความเหมาะสมทางวัฒนธรรม การใช้งานจริงในเวิร์กช็อป และความเข้าใจง่ายสำหรับผู้ทำแบบประเมิน ทั้งในรูปแบบออนไลน์และเอกสารกระดาษ

## แบบสอบถามฉบับสั้น (24 ข้อ)

### คำชี้แจงผู้ทำแบบประเมิน

เลือกคำตอบเพียง 1 ข้อในแต่ละคำถามที่ “ใกล้เคียงคุณที่สุด” ในสถานการณ์ทั่วไป (ไม่ใช่วันที่เครียดสุดหรือดีสุด) ไม่มีถูกผิด

### แบบประเมิน (ผู้ทำเห็นเฉพาะส่วนนี้)

1. เริ่มกะแล้วงานค้างเยอะ คุณมักจะ…
   - A จัดลำดับงานเร่งด่วนแล้วกำหนดเป้าหมายชัด ๆ ให้ทีม
   - B ชวนคุยให้ทีมฮึกเหิม แล้วเริ่มลงมือแบบ “ไปด้วยกัน”
   - C ช่วยประสานให้ทุกคนทำงานลื่นไหล ลดความตึงเครียด
   - D ตรวจรายการ/ข้อมูลให้ครบก่อน เพื่อไม่ให้พลาด
2. ญาติผู้ป่วยไม่พอใจเสียงดัง คุณมัก…
   - A พูดตรงประเด็น ตั้งกรอบการคุย และตัดสินใจแนวทางทันที
   - B ใช้น้ำเสียงเป็นมิตร สร้างบรรยากาศให้คลี่คลายก่อน
   - C รับฟังใจเย็น ประคองอารมณ์ และค่อย ๆ พาไปสู่ทางออก
   - D ขอข้อมูล/ข้อเท็จจริงทีละส่วน แล้วอธิบายตามหลักการ
3. ต้องประสานงานหลายหน่วยพร้อมกัน คุณมัก…
   - A เลือกงานคอขวดก่อน แล้วเร่งให้จบเป็นช่วง ๆ
   - B โทร/คุยให้ครบคน ทำให้ทุกฝ่าย “ร่วมมือ” ได้เร็ว
   - C คอยตามงานแบบสม่ำเสมอ ไม่ให้ใครรู้สึกถูกกดดัน
   - D ทำเช็กลิสต์/เอกสารกำกับให้ชัด ลดความคลาดเคลื่อน
4. ในที่ประชุมทีม คุณมัก…
   - A เสนอข้อสรุปและทางเลือกที่ตัดสินใจได้
   - B ชวนทุกคนมีส่วนร่วม เติมพลัง/สร้างบรรยากาศ
   - C ฟังให้รอบด้าน เน้นความกลมกลืนและความร่วมมือ
   - D ตั้งคำถามเชิงเหตุผล ตรวจความเสี่ยง/รายละเอียด
5. มีแนวปฏิบัติใหม่ประกาศใช้กะทันหัน คุณมัก…
   - A ปรับแผนทันที และบอกทีมว่า “เอาแบบนี้” เพื่อเดินต่อ
   - B ชวนคุยให้ทีมเปิดใจ แล้วค่อยพาไปลองทำ
   - C ให้เวลาเพื่อนร่วมงานปรับตัว และช่วยกันค่อย ๆ เปลี่ยน
   - D ขอเอกสาร/หลักฐานอ้างอิง ทำความเข้าใจให้ชัดก่อน
6. งานเอกสาร/บันทึกการพยาบาลหนาแน่น คุณมัก…
   - A ทำส่วนที่กระทบงานภาพรวมก่อน แล้วค่อยเก็บรายละเอียด
   - B ทำไปคุยไป ขอความร่วมมือ/แบ่งงานให้เสร็จไว
   - C ทำแบบคงเส้นคงวา สม่ำเสมอ ไม่เร่งจนเครียด
   - D เน้นความถูกต้อง รูปแบบ และมาตรฐานครบถ้วน
7. เกิด near miss ด้านยา คุณมัก…
   - A สั่งหยุด/แก้ไขทันที แล้วจัดการระบบป้องกันซ้ำ
   - B สื่อสารให้ทีมไม่ตื่นตระหนก แล้วชวนช่วยกันเรียนรู้
   - C ดูแลความรู้สึกคนที่เกี่ยวข้อง ลดการโทษกัน
   - D ไล่ลำดับเหตุการณ์ หา root cause จากข้อมูล
8. ต้องให้ feedback เพื่อนร่วมงานที่ทำพลาด คุณมัก…
   - A พูดตรง ๆ ว่าควรปรับอะไร และคาดหวังผลลัพธ์อะไร
   - B เริ่มจากชมก่อน แล้วค่อยเสนอให้ปรับแบบเป็นกันเอง
   - C เลือกคุยส่วนตัวอย่างนุ่มนวล เพื่อรักษาความสัมพันธ์
   - D ยกตัวอย่างมาตรฐาน/หลักฐาน เพื่อให้เห็นภาพชัด
9. ผู้รับคำปรึกษาเงียบ ไม่ค่อยพูด คุณมัก…
   - A ถามคำถามชัด ๆ เพื่อให้ได้ประเด็นและเดินต่อ
   - B ใช้บทสนทนาเบา ๆ เพื่อเปิดใจและสร้างความไว้วางใจ
   - C รอจังหวะ ให้พื้นที่ และรับฟังอย่างอดทน
   - D สรุปสิ่งที่ได้ยินเป็นข้อ ๆ เพื่อช่วยจัดระเบียบความคิด
10. ทำงานกลุ่มนอกงาน (เช่น โครงการ/กิจกรรม) คุณมัก…
    - A รับบทนำ กำหนดไทม์ไลน์และเป้าหมาย
    - B เป็นคนเชื่อมคน ทำให้ทีมคึกคักและเดินหน้า
    - C เป็นคนคอยประสาน ช่วยให้ทุกคนทำงานเข้ากันได้
    - D เป็นคนวางระบบ ตรวจคุณภาพและความถูกต้อง
11. มีเคสฉุกเฉินหลายเคสเข้ามาพร้อมกัน คุณมัก…
    - A ตัดสินใจเร็ว จัดลำดับความสำคัญ แล้วสั่งการ
    - B สื่อสารเร็ว ชัด และกระตุ้นทีมให้ทำงานพร้อมกัน
    - C คงความนิ่ง ประคองทีมให้ไม่แตกตื่น
    - D ยึดตามแนวทาง/โปรโตคอล ตรวจความถูกต้องทุกขั้น
12. เมื่อมีความขัดแย้งในทีม คุณมัก…
    - A เคลียร์ให้จบ โดยโฟกัสทางออกและข้อตกลง
    - B คุยให้เข้าใจกัน ใช้อารมณ์เชิงบวกนำ
    - C ลดความตึงเครียด ค่อย ๆ พาคืนสู่ความร่วมมือ
    - D ใช้ข้อเท็จจริง/กติกา ลดการถกเถียงเชิงอารมณ์
13. ต้องนำเสนอผลงานต่อผู้บริหาร คุณมัก…
    - A สรุปผลลัพธ์/ตัวชี้วัด ชัด กระชับ
    - B เล่าเรื่องให้เห็นภาพ ดึงความสนใจ
    - C ขอบคุณทีมและเน้นความร่วมมือที่ทำให้สำเร็จ
    - D เตรียมข้อมูล/ตัวเลข/เอกสารอ้างอิงครบ
14. เมื่อถูกเร่งงาน คุณมัก…
    - A เร่งตัดสินใจ ลดสิ่งไม่จำเป็น
    - B ขอแรงคนอื่น ชวนช่วยกันให้เสร็จ
    - C ทำไปทีละส่วน รักษาจังหวะไม่ให้หลุด
    - D เพิ่มการตรวจทานเพื่อกันพลาด
15. หากต้องเลือกซื้อของราคาแพง (เช่น มือถือ/เครื่องใช้) คุณมัก…
    - A เลือกตัวที่ตอบโจทย์หลัก ๆ แล้วตัดสินใจเร็ว
    - B เลือกตัวที่ “ใช้แล้วมีความสุข/ภาพลักษณ์ดี”
    - C เลือกตัวที่คุ้ม ใช้ได้นาน ไม่จุกจิก
    - D เปรียบเทียบสเปก/รีวิวละเอียดก่อนซื้อ
16. วางแผนทริปกับเพื่อน คุณมัก…
    - A กำหนดเป้าหมายทริป แล้วจัดตารางให้คุ้ม
    - B เน้นกิจกรรมสนุก ๆ และคนในทริป
    - C เน้นความสบาย ทุกคนโอเค ไม่มีใครอึดอัด
    - D วางแผนเส้นทาง/งบ/ความเสี่ยงอย่างละเอียด
17. เมื่อมีงานซ้ำ ๆ ยาว ๆ คุณมัก…
    - A หาทางทำให้เร็วขึ้น หรือทำให้จบเป็นช่วง
    - B หาเพื่อนคุย/ทำร่วมกันให้ไม่น่าเบื่อ
    - C ทำอย่างสม่ำเสมอ จนครบตามแผน
    - D ปรับขั้นตอนให้มาตรฐานและผิดพลาดน้อยที่สุด
18. ได้รับมอบหมายงานใหม่ที่ไม่ถนัด คุณมัก…
    - A ลงมือก่อน แล้วเรียนรู้ระหว่างทาง
    - B ขอคำแนะนำ พูดคุยแลกเปลี่ยนกับคนเก่ง ๆ
    - C ขอเวลาและการสนับสนุน ค่อย ๆ ฝึกจนมั่นใจ
    - D ศึกษาคู่มือ/มาตรฐานให้เข้าใจ ก่อนเริ่มจริง
19. ต้องสอนรุ่นน้อง/นักศึกษา คุณมัก…
    - A สอนให้ทำได้จริง เน้นจุดสำคัญและผลลัพธ์
    - B สร้างแรงบันดาลใจ ให้กล้าถามกล้าลอง
    - C สอนแบบใจเย็น ให้กำลังใจ และค่อย ๆ ปรับ
    - D สอนตามขั้นตอน มีเหตุผลและเกณฑ์ชัดเจน
20. เวลาคิดแก้ปัญหา คุณมัก…
    - A มอง “ทางชนะ” และตัดสินใจจากผลลัพธ์
    - B มองคนและการยอมรับ เพื่อให้เดินต่อได้
    - C มองความต่อเนื่องและความสัมพันธ์ในทีม
    - D มองความถูกต้อง หลักฐาน และความเสี่ยง
21. ถ้ามีคนโต้แย้งคุณต่อหน้า คุณมัก…

    - B ใช้อารมณ์เชิงบวก เบี่ยงให้คุยแบบสร้างสรรค์
    - C พยายามลดความแรง เพื่อไม่ให้บานปลาย
    - D ขอหลักฐาน/ข้อมูล แล้วค่อยอภิปรายตามข้อเท็จจริง
22. เมื่อทำงานร่วมกับคนที่ช้า/ลังเล คุณมัก…
    - A เร่งให้ตัดสินใจ และเสนอทางเลือกชัด ๆ

    - C ให้เวลาและช่วยประคอง ไม่กดดัน
    - D อธิบายขั้นตอน/เหตุผล เพื่อให้มั่นใจ
23. เมื่อเจอคนทำงาน “ไม่เป็นระบบ” คุณมัก…
    - A จัดใหม่ให้เห็นผลเร็ว
    - B ชวนเขาคุยให้ร่วมมือ แล้วค่อยปรับ
    - C ปรับทีละนิด ไม่ให้เขารู้สึกแย่
    - D วางมาตรฐาน/เช็กลิสต์ และยืนยันให้ทำตาม

    - A เด็ดขาด ตรง เป้าหมายชัด
    - B เข้าถึงง่าย ชวนคุย สร้างพลัง
    - C ใจเย็น อดทน ดูแลคนรอบข้าง
    - D รอบคอบ เนี๊ยบ ใช้เหตุผล

## วิธีคิดคะแนน (สำหรับระบบ/ผู้จัดเวิร์กช็อป)

### หลักการให้คะแนน

คำตอบแต่ละข้อสอดคล้องกับ 1 มิติ (D, i, S, C) ให้นับคะแนนสะสมเป็นคะแนนดิบ (raw score)

### กุญแจเฉลยการนับคะแนน (Answer Key)

ทุกข้อใช้รูปแบบเดียวกัน: A = D, B = i, C = S, D = C (ข้อที่ 1–24)

## วิธีแปลผล (Interpretation Rules)

### 1) เลือกสไตล์หลักและรอง

- สไตล์หลัก (Primary) = คะแนนสูงสุด
- สไตล์รอง (Secondary) = คะแนนอันดับ 2
- ถ้าคะแนนสูงสุดเสมอกัน ให้รายงานเป็น “สไตล์ผสม” (เช่น D/i)

### 2) ระดับความเด่นชัด (Intensity)

- 0–5 = ต่ำ
- 6–9 = ปานกลาง
- 10–24 = สูง

ใช้เพื่อสะท้อนตนเอง ไม่ใช่มาตรฐานเชิงคลินิก

### 3) ข้อควรระวังในการสรุปผล

- คนส่วนใหญ่เอน 1–2 โซน ไม่ได้มีแค่แบบเดียว
- ผลสะท้อนแนวโน้มพฤติกรรมที่เลือก ไม่ใช่ความสามารถเฉพาะงาน
- ห้ามใช้ผลเพื่อ “ตีตรา” หรือจัดลำดับคุณค่า

## คำอธิบาย 4 มิติ + แนวทางใช้ในงานพยาบาล/การให้คำปรึกษา

นิยามสั้น ๆ:

- D มักตรง เร็ว เน้นผลลัพธ์
- i มักเปิดเผย ชวนคุย สร้างพลัง
- S มักใจเย็น อดทน เน้นความร่วมมือ
- C มักวิเคราะห์ รอบคอบ เน้นมาตรฐาน/ความถูกต้อง

### D (Dominance) สูง

เด่น: ตัดสินใจเร็ว กล้ารับผิดชอบ เหมาะกับสถานการณ์ฉุกเฉิน/งานที่ต้อง “ปิดงาน”

เสี่ยง: สื่อสารตรงเกินไป กดดันคนอื่น เผลอมองข้ามอารมณ์คน

ทริค: ระวัง “รีบแก้ปัญหาแทน” ใช้คำถามปลายเปิดก่อนให้คำแนะนำ

### i (Influence) สูง

เด่น: สร้างสัมพันธภาพเก่ง ดึงความร่วมมือ เหมาะกับงานสื่อสาร/การเปลี่ยนแปลง

เสี่ยง: เผลอข้ามรายละเอียด ตัดสินใจจากความรู้สึก/บรรยากาศ

ทริค: ใช้โครง “สรุป-ตกลงแผน-ติดตาม” เพื่อไม่หลุดประเด็น

### S (Steadiness) สูง

เด่น: ใจเย็น สม่ำเสมอ ดูแลคนรอบข้าง เหมาะกับงานต่อเนื่อง/ดูแลระยะยาว

เสี่ยง: หลีกเลี่ยงความขัดแย้ง ไม่กล้าปฏิเสธ ช้าในการเปลี่ยนแปลง

ทริค: ฝึก “ขอบเขต (boundary)” และประโยคยืนยันสิทธิอย่างสุภาพ

### C (Conscientiousness) สูง

เด่น: รอบคอบ แม่นมาตรฐาน ลดความเสี่ยง เหมาะกับงานคุณภาพ/ความปลอดภัย

เสี่ยง: คิดนาน เพอร์เฟกชันนิสม์ วิจารณ์ตัวเอง/คนอื่นสูง

ทริค: เพิ่มการสะท้อนความรู้สึก (ไม่ยึดแต่ข้อมูล) และกำหนด “พอแล้วที่ดีพอ”

## การแปลผลแบบคู่สไตล์ (Top 2)

- D/i: มุ่งผล + โน้มน้าวเก่ง (ขับเคลื่อนทีมเร็ว) ระวังพูดเร็ว/แรง
- i/D: เข้าหาคน + กล้าชน (นำแบบมีเสน่ห์) ระวังใจร้อน/ตัดสินเร็ว
- D/C: เด็ดขาด + มาตรฐานสูง (ยืนบนเหตุผล) ระวังเคร่งและคาดหวังสูง
- C/D: วิเคราะห์ + กล้าตัดสินเมื่อมั่นใจ (คมและชัด) ระวังช้าในช่วงข้อมูลไม่ครบ
- D/S: เร็วแต่ยังคุมความนิ่งได้ (จัดระบบคน) ระวังสลับโหมดเร็ว-ช้าให้ทีมงง
- S/D: ใจเย็นแต่ยืนหลักได้ (ประคองแล้วค่อยตัดสิน) ระวังเก็บอารมณ์จนระเบิด
- i/S: เป็นมิตร อบอุ่น ประสานทีมเก่ง (เหมาะงานคน) ระวังเลี่ยงความขัดแย้ง
- S/i: สุขุมแต่เข้าถึงง่าย (ดูแลคนแบบนุ่ม) ระวังไม่กล้าพูดสิ่งยาก
- i/C: สื่อสารดี + มีเหตุผล (พรีเซนต์ข้อมูลเก่ง) ระวังเครียดเมื่อรายละเอียดไม่พร้อม
- C/i: ข้อมูลแน่น + คุยรู้เรื่อง (อธิบายเก่ง) ระวังวิจารณ์มากไปในเวลาจำกัด
- S/C: มั่นคง + รอบคอบ (งานคุณภาพสม่ำเสมอ) ระวังช้าเมื่อมีแรงกดดัน
- C/S: ระเบียบ + ใจเย็น (มาตรฐานและความปลอดภัย) ระวังต้านการเปลี่ยนแปลง
